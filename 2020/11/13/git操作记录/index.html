gitlab如何同步fork后的项目

#添加项目A的远程仓库地址到upstream 
git remote add upstream <你朋友项目A的仓库地址> 
#把项目A的更新来到本地的upstream里 
git fetch upstream 
#切换到你自己想要merge的分支，我这里用master 
git checkout master 
#merge项目A的更新到你的branch 
git merge upstream/master

git rebase使用

git checkout master
git pull
git checkout local
git rebase -i HEAD~2  //合并提交 --- 2表示合并两个
